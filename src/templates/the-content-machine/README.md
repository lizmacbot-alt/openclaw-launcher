# The Content Machine — OpenClaw Agent Template

> Turn your OpenClaw agent into a relentless content engine that researches, writes, schedules, and optimizes across every platform.

## What's Inside

A complete agent workspace pre-configured for content creators, marketers, and solopreneurs who want to 10x their output without burning out.

### Files Included

| File | Purpose |
|------|---------|
| `SOUL.md` | Content strategist personality — creative, data-driven, brand-aware |
| `IDENTITY.md` | Your agent's content creator identity |
| `USER.md` | Template for your profile, brand voice, and target audience |
| `AGENTS.md` | Workflow rules for content ops — research, write, schedule, analyze |
| `HEARTBEAT.md` | Auto-pilot tasks: trending topics, scheduled posts, engagement tracking |
| `TOOLS.md` | Setup notes for social platforms, analytics, and writing tools |
| `MEMORY.md` | Content calendar, performance history, audience insights |
| `CONTENT-SYSTEM.md` | The full content pipeline — ideation → creation → distribution → analysis |

## What It Does

- **Research trending topics** in your niche automatically
- **Write posts, threads, newsletters, and blog articles** in your brand voice
- **Maintain a content calendar** and remind you what's due
- **Repurpose content** across platforms (tweet → thread → blog → newsletter)
- **Track what works** — saves performance data and optimizes future content
- **Monitor competitors** and surface content gaps you can fill
- **Draft and queue posts** for your review before publishing

## Setup (5 minutes)

1. Copy all `.md` files to your OpenClaw workspace (`~/.openclaw/workspace/`)
2. Edit `USER.md` with your brand info, voice, and platforms
3. Edit `TOOLS.md` with your API keys and account handles
4. Start chatting — your agent already knows the workflow

## Platforms Supported

Works with any platform your OpenClaw has access to:
- Twitter/X, LinkedIn, Reddit, Instagram captions
- Blog posts (any CMS), newsletters, YouTube scripts
- Email marketing copy, landing page copy

## Pro Tips

- Fill out `USER.md` thoroughly — the more brand context, the better the output
- Let the heartbeat run for a few days — it learns your niche fast
- Use "repurpose mode" to turn one piece into 5+ platform-specific posts
- Check `MEMORY.md` weekly — your agent tracks what performs best
