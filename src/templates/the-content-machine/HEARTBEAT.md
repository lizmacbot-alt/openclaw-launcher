# HEARTBEAT.md — Content Auto-Pilot

## Every Heartbeat (check in order)

### 1. Content Calendar
- Check `MEMORY.md` for today's scheduled content
- If something is due, draft it and notify the user
- If nothing is due, check if there's a gap in the schedule

### 2. Trending Topics
- Search for trending topics in the user's niche (web search)
- If something relevant is trending, draft a reactive post
- Save trend notes in `memory/YYYY-MM-DD.md`

### 3. Performance Check
- If the user has posted recently, note engagement numbers
- Update performance tracking in `MEMORY.md`
- Identify any breakout content worth repurposing

### 4. Content Pipeline
- Review drafts in progress — anything ready to finalize?
- Generate 1-2 new content ideas based on recent trends
- Add to the idea backlog in `MEMORY.md`

### 5. Competitor Watch
- Check competitor accounts listed in `USER.md`
- Note any content strategies worth adapting
- Save observations in daily notes

## When to Alert the User
- Content is due in the next 2 hours and not yet approved
- A post is performing exceptionally well (repurpose opportunity)
- A trending topic is highly relevant and time-sensitive
- The content calendar has gaps in the next 3 days

## When to Stay Quiet
- Late night / early morning (respect timezone)
- All scheduled content is handled
- No significant trends or performance changes
- Last check was < 30 minutes ago
