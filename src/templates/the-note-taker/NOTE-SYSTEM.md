### Phase 2: Organize (Background)

**Goal:** Structure emerges from your notes naturally.

**What the agent does automatically:**
- Tags each note by type and topic
- Files in the correct daily note
- Links to related previous notes
- Tracks recurring themes in MEMORY.md
- Moves actionable items to the task list

**What you do:**
- Nothing. Seriously. The agent handles this.
- (Optionally) tell the agent to re-tag or reorganize if something feels wrong

### Phase 3: Review (Periodic)

**Goal:** Turn raw captures into useful knowledge.

**Daily (evening, optional):**
- Agent offers a journal prompt
- Quick scan of today's captures
- Any loose tasks to close out?

**Weekly (Sunday or Monday):**
- Agent compiles the week's notes
- Highlights top themes and ideas
- Lists open tasks
- Surfaces connections you might have missed
- Saves a weekly review file

**Monthly (first week):**
- Review weekly summaries
- Update MEMORY.md with lasting insights
- Archive or close stale ideas
- Celebrate: look how much you captured!

## Tips for Getting the Most Out of This

1. **Lower your bar.** Not every note needs to be profound. Capture the mundane too.
2. **Voice notes work.** If your OpenClaw has TTS/STT, just talk.
3. **Review weekly.** This is where 90% of the value comes from.
4. **Trust the system.** Your agent remembers so you don't have to.
5. **Connect the dots.** Ask your agent "what have I been thinking about lately?"
