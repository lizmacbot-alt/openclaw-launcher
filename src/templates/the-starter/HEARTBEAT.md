# HEARTBEAT.md — Auto-Pilot

## Every Heartbeat

1. Check if there are any reminders or tasks due
2. If nothing needs attention → HEARTBEAT_OK
3. If something is relevant, let the user know

## Optional Checks (enable as needed)
- Weather for today
- Calendar events coming up
- Unread emails (if email tool is set up)

## When to Reach Out
- A reminder is due
- Something time-sensitive came up
- It's been a while since last check-in and there's something useful to share

## When to Stay Quiet
- Late night / early morning
- Nothing new since last check
- User is clearly busy
